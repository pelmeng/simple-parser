# 📚 Book Parser (Python)

Скрипт на Python для автоматического сбора информации о книгах с сайта
http://books.toscrape.com

Парсер проходит по страницам каталога и собирает данные о книгах, после чего сохраняет их в Excel файл.

---

## ⚙️ Функции

Скрипт автоматически:

* открывает страницы каталога
* собирает информацию о книгах
* извлекает название, цену и наличие
* сохраняет данные в Excel файл

---

## 📊 Какие данные собираются

Для каждой книги собирается:

* **Название**
* **Цена**
* **Наличие**

Пример результата:

| Название             | Цена   | Наличие  |
| -------------------- | ------ | -------- |
| A Light in the Attic | £51.77 | In stock |
| Tipping the Velvet   | £53.74 | In stock |

---

## 🛠 Используемые технологии

* Python
* requests
* BeautifulSoup
* pandas

---

## 📦 Установка

Установите необходимые библиотеки:

pip install requests beautifulsoup4 pandas

---

## ▶️ Запуск

Запустите скрипт:

python WebParsing.py

После выполнения появится файл:

books.xlsx

---

## 🔧 Настройки

В коде можно изменить диапазон страниц:

START_PAGE = 1
END_PAGE = 50

Например:

END_PAGE = 10

---

## 📁 Результат

После работы скрипта создается Excel файл:

books.xlsx

В нем будут собраны все книги со страниц сайта.

---

## ⚠️ Примечание

Сайт http://books.toscrape.com используется специально для обучения веб-парсингу.
